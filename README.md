### Cài Đặt

1. Clone the repo
```sh
git clone https://github.com/vacnex/Hotel-Manager-Project.git
```
2. Cập nhật các gói Nuget đi kèm trong project
```sh
HotelManager
```
4. Đổi API url trong `MainWindow.xaml`
```C#
string URL_NAME = 'Nhập API Link';
```
